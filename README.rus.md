# Decache (русская версия)

**Decache** — это специализированный сканер кэша веб-браузеров для Windows XP и новее.  
Все найденные файлы надёжно сравниваются с метаданными утерянных медиа, в первую очередь — удалённых видео с YouTube.

## Использование из командной строки

Вы можете указать путь к папке с кэшем (или ко всему диску) прямо в аргументе командной строки, чтобы не выбирать её вручную в проводнике. Например:

`start_decache.bat "E:\Backups\Old Laptop"`

Также можно просканировать несколько компьютеров, передав текстовый файл, где каждая строка содержит путь:

`start_decache.bat computers.txt`

### Дополнительные ключи командной строки

- `start_decache.bat /keepall` — сохранить **все** неопознанные видео в папку «Unverified» (непроверенные).
- `start_decache.bat /silence:1` — игнорировать все ошибки, не предпринимать никаких действий.
- `start_decache.bat /silence:2` — игнорировать ошибки, но при попытке доступа к запрещённой папке один раз попробовать получить права владельца.

Все ключи можно комбинировать.
